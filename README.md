# Installation
Please add this URL to your repositories under `Plugins > Manage Repositories > New Repository`:
```
https://raw.githubusercontent.com/moeux/FinTrim/main/manifest.json
```
Search for FinTrim and click on install.

> [!IMPORTANT]  
> There will be no cleanup unless you specify what to clean up under the FinTrim settings.
> Please do so under `Plugins > Installed > FinTrim > Settings`
